^Slanted edge

Perhaps viewing this image will give you a hint on how to have a slanted-edge graphical bar:

Here, the numbers to the left of each tile indicates an index of the tile array, while the numbers on the right are the fill amount. Combine this image with the one below:

and you get the idea on how it works. Technically, the amount of fill to be converted into tile index is exceeding the number of pieces to accommodate the next tile's fill amount, the tiles shown in the red boxes shows that you have multiple full tiles. Also notice that since there is no next tile after the right end, the number of pieces can safely be set to the number of distinct fill amounts.
