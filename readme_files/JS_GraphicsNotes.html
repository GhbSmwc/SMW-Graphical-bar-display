Notes:
  • The “Fill table to use”'s <Number>/<Number>/<Number> denotes the number of pieces for left end, each middle, and right end. Stuff like the megaman-styled and diagonal/outlined/rounded fill edges uses a different piece maximums. This means you need to change these defines (in GraphicalBarDefines/StatusBarSettings.asm): 
    	!Default_LeftPieces                  = 3             ;\These will by default, set the RAM for the pieces for each section
	!Default_MiddlePieces                = 8             ;|
	!Default_RightPieces                 = 3             ;/
